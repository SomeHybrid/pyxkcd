Welcome to pyxkcd's documentation!
==================================

pyxkcd is a simple, easy-to-use library wrapping xkcd's api.

.. note::
   This library is in no way affiliated with xkcd or Randall Munroe.

.. warning:: 
   For that one person using this library, this is still in early development.
   There probably will be breaking changes every other second. In fact, the version
   this documentation came out has a breaking change.

Features
--------
- Asynchronous support
